JavaScript - Increment & Decrement Operator

The increment (++) and decrement (--) operators work differently, depending on how they are used.
You can use them as either prefix or postfix to a variable, which decides when the operation happens.

Prefix

When you use the operator as prefix to a variable name, the operation happens "before" other operations.
That means if the operator is used in an expression, the operation happens before the other operations in the expression.

Postfix

When you use the operator as postfix to a variable name, the operation happens "after" other operations.
That means if the operator is used in an expression, the operation happens after the other operations in the expression.
