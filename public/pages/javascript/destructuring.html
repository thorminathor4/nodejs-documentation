JavaScript - Destructuring

You can destructure the values of an object or an array into variables.
When you destructure, you take one ore more values from an array and intialize variables with those values


This is the simplest way to destructure:
  • Object:
    (The value of the "name" property of the object, which is the string "John", is assigned to the variable "name"):

    const {name} = {name: "John", age: 22};

  • Array:
    (The first value in the array, which is the string "apple", is assigned to the variable "fruit"):

    const [fruit] = ["apple", "banana", "pineapple"];
