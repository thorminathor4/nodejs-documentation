JavaScript - Spread Operator

You can use the spread operator to spread the content of an array or object into another array or object.
The notation is three dots (...) as a prefix to the array or object with the content you want to spread.
It must be used inside a literal array or object expression.

Spread Array Content

Here the elements of the array "fruits" is spread into a new array which the variable "fruitsCopy" gets initialized to:

const fruits = ["apple", "banana", "pineapple"];
const fruitsCopy = [...fruits];

Spread Object Content

Here the properties of the object "person" is spread into a new object which the variable "personCopy" gets initialized to:

const person = {name: "John", age: 22};
const personCopy = {...person};
