Express - Create a Server

This is a guide on how to make an express server.
To access a server there must be routes that can be accessed via endpoints.
To see how to make routes that can send back web-pages and other data, go to this guide: Create a Route.

  1. Make a folder for the project
  2. Install the "express" module. See how to install modules here: NPM & Projects.
  3. Create the file "app.js" in the project folder and create the server by writing the following in "app.js":
    1. Import the "express" module and assgn the recieved object to a variable you with the name "express".
      See how to import modules here: Importing & Exporting Modules.
    2. Create the server object with the name "app":

      const app = express();

    3. This is where you should one or both of the following:
      • Define routes by using methods on the server object.
      • Import a router/routers and use it/them on the server object.
      To see how to make routes that can send back web-pages and other data, go to this guide: Create a Route.
      To see how to make a router, go to this guide: Router.
    4. Define the port where: 0 ≤ PORT ≤ 65535 (Recommended ports for development are 3000, 5000 or 9000):

      const PORT = 3000;


      You can also allow for a costom port to be set from an environment variable.
      (Here the "PORT" will be initialized to the environment variable "PORT" if provided and 3000 if not provided):

      const PORT = process.env.PORT || 3000;


      Read more about environment variables here: Use Environment Variables.
    5. Use the listen method on the server object to start the server on the port:

      app.listen(PORT);


      You can also define a callback that is called when the server starts:

      app.listen(PORT, () => {
          console.log("Server started on port:", PORT);
      });


You can start the server by executing the file "app.js" that you just made.
See how to execute a file here: Basics of Node.js.

The domain of the server will be "IP:PORT" where:
  • IP = The IP-Address of the hosting computer
  • PORT = The port of the server
On the computer hosting the server, you can just use "127.0.0.1" or "localhost" instead of the IP-Address.

To see how to make routes that can send back web-pages and other data, go to this guide: Create a Route.
To see how to make a router, go to this guide: Router.
