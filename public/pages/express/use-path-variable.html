Express - Use Path Variable

You can include a path variable in the path of a route.
The path consist of zero or more path segments seperated by slashes fx. "/users/1".
You can define a path to include one or more path variables.
This is done by specifying one or more of the segments as a path variable.
In express you can define a path variable in a path by writing colon followed by the variable name in the path fx "/users/:id".
The path "/users/:id" has one path variable with the name "id".

Here is a guide how to use a path variable:

  1. Define the a path with one or more path variables for the route.
    (Here a path for a get method is defined with one path variable called "id"):

    app.get("/users/:id", (req, res) => {

    });

  2. The path variable(s) will be properties of the params object inside the request object.
    (Here we declare a variable "id" to be the the path varaiable "id", by assigning it to the id property):

    app.get("/users/:id", (req, res) => {
        const id = req.params.id;
    });


    The path variable(s) will be a string/strings, so if another type is needed, it/they must be converted.
    (Here the the path variable "id" is converted to a number):

    app.get("/users/:id", (req, res) => {
        const id = Number(req.params.id);
    });
