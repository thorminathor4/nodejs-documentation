Express - Send a File

This is a guide on how to send files on a route on an express server.
To see how to make an express server, go to this guide: Create a Server.
To see how to make a route, go to this guide: Create a Route.

  1. To send a file, use the sendFile method on the response object:

    app.get("/", (req, res) => {
        res.sendFile(__dirname + "/public/index.html");
    });
