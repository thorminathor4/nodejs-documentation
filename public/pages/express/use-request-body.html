Express - Use Request Body

You can have a body in an HTTP request.
In Express the request body will by default be repressented as a Buffer of the bytes inside the body.
You can make an express server convert/parse all the request bodies it recieves into another format.
To see how to make an express server, go to this guide: Create a Server.
See how to make routes here: Create a Route.


This is a giude on, how to use the request body:
  1. Make the express server parse the request bodies as *JSON (convert them from *JSON to objects) with this method:

    app.use(express.json());

  2. Make a route for a post, put or a patch method
    (Here is a route for a post method with the path "/users"):

    app.post("/users", (req, res) => {

    });

  3. The request body will be an object in the "body" property of the request object
    (Here the variable "body" is initialized to the body object):

    app.post("/users", (req, res) => {
        const body = req.body;
    });

*JSON = JavaScript Object Notation
