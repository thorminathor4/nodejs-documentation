Express - Use Query String

You can include a query string in a url.
This is done by writing a question mark after the path.
Then one or more keys followed by an equals sign and a value can be written, seperated by an ampersand fx. "/users?name=John&age=22"
That query string then contain the following:

  • The key "name" with the value "John"
  • The key "age" with the value "22"
To see how to make an express server, go to this guide: Create a Server.
See how to make routes here: Create a Route.

Here is how to use the query string:

The key(s) of the query string will be properties of the query object inside the request object.
The keys will have their value specified in the query string.
(Here we declare a variable "name" to be the the path varaiable "name", by assigning it to the name property):

app.get("/users", (req, res) => {
    const name = req.query.name;
});


The value(s) in the query string will be a string/strings, so if another type is needed, it/they must be converted.
(Here the value of the key "age" is converted to a number):

app.get("/users", (req, res) => {
    const age = Number(req.query.age);
});
