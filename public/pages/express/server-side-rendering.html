Express - Server Side Rendering

Server side rendering is when a page is rendred on the server, before it is sent to the client.
The pages can pontentially be dynamically rendered, meaning they will can have different content every time they are sent to a client.


This guide will show different ways to render a page on the server.

Load Files

The examples below will show how to render pages by handling strings.
You can use the "fs" module to load the contents of files into strings.
Read more about that here: File System

Concatenate Content

You can render a page by concatenating content like this
(Here the frontpage is rendered by concatenating: a header, the frontpage content "frontpageContent" and a footer):

const frontpage = header + frontpageContent + footer;

Replace Keywords

You can also render a page by replacing a keyword like this
(Here the frontpage is rendered by replacing the keyword "%%NAVBAR%%" with the navbar content in the frontpageTemplate):

const frontpage = frontpageTemplate.replace("%%NAVBAR%%", navbar);
