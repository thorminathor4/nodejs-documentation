Express - Redirection

You can redirect an HTTP response from a route to another endpoint with an express server.
To see how to make an express server, go to one of this guide: Create a Server.
See how to make routes here: Create a Route.


To redirect a response, use the redirect method on the respone object and give it the endpoint/path to redirect to as a string
(This this will redirect the response from the route that recieves a get request to the route of the endpoint "/frontpage"):

res.redirect("/frontpage");
