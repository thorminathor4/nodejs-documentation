Node.js - Nodemon

Nodemon is a module for Node.js.
It can moniter your project and rerun it automatically, when the project has been edited.
This makes it a useful tool for development.

Install Nodemon

It is typical to install Nodemon globally.
See how to install modules here: NPM & Projects.
When you have Nodemon installed globally, you can also just moniter one file.

Moniter A Project/File

When you have installed nodemon, you can start nodemon using the following command.
Nodemon will then moniter the file/project and re-run the specified file upon a chang to the file/project.
(In this case, the file "app.js" will be continually re-executed):
Windows:

nodemon.cmd app.js

Mac:

nodemon app.js
