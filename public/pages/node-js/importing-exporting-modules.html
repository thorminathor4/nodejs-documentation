Node.js - Importing & Exporting Modules

Importing Modules

Modules can be imported, by using the built in require function


File-Path:
When you give a file-path in a string as parameter, you can import a module from a JavaScript file that exports a module:
(Here a router is imported from the file "router.js" in the directory "routers"):
const router = require("./routers/router.js");
To see how to make a express router go to this guide: Router.

Module Name:
When you give a module name in a string as parameter, you can import a named module.
See how to install modules here: NPM & Projects.
(Here is how the "express" module can be imported):
const express = require("express");
To see how to make an express server, go to this guide: Create a Server.

When you have imported a module, it is typically an object with methods and possibly other properties.

Exporting Modules

Modules can be exported, by using the exports property of the built in module object
(Here an object with a method called "getMessage" that returns "Hello World!" is exported):
module.exports = {
    getMessage: () => "Hello World!"
};
You can basically export anything, but it is good practice to export an object with methods and optionally other properties.
It is common to have methods in the exported object.
