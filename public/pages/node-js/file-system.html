Node.js - File System

In Node.js you can interact with the file system on the computer, using the a module called "fs".
See how to import modules here: Importing & Exporting Modules.
Typically the "fs" module is loaded into a variable with the name "fs".

Read File

To read a file synchronously, you can use the readFileSync method and give it the file path as a string
(Here the file "file.txt" is read into the variable "file"):

const file = fs.readFileSync("./file.txt");



The readFileSync method returns a Buffer object representing the bytes in the file.
If the file contains text format data, the buffer can be converted to a string.
(File extentions that contain text fortmat data include: txt, html, css, svg, xml, json, js, java, c, cpp, cs):

const file = fs.readFileSync("./file.txt").toString();
