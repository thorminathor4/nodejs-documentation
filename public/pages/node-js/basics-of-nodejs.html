Node.js - Basics of Node.js

Node is a JavaScript runtime envionment, that can execute JavaScript.
The name Node.js just clarifies that it is the JavaScript runtime envionment being referred to.

Download Node.js

You can download the Node.js installer here: https://nodejs.org/en/download/.

Install Node.js

When you have dowloaded the installer, you can install Node.js.

Execute A JavaScript File

When you have installed Node.js, you can start executing JavaScript files.
You can execute a JavaScript file from a terminal with the "node" command
(Here the file "app.js" is executed):
node app.js
If you use the log method on the console object in the file, the text will show in the terminal, where the file has been executed.
(For example if you log "Hello World!" like this):
console.log("Hello World!");
