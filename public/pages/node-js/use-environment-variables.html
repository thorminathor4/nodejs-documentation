Node.js - Use Environment Variables

You can use environment variables in Node.js.

Here is how you can run a file with an environment variable set
(Here the environment variable "PORT" is set to 3000 and the file "app.js" is executed):

PORT=3000 node app.js


This only works on Mac though, but the "cross-env" module can "fix" that.
Read more about the "cross-env" module here: Cross Environment.

When you execute a file, the the environment variable(s) will be properties of the "env" object inside the "process" object.
(Here the environment variable "PORT" is assigned to the variable "PORT"):

const PORT = process.env.PORT;
