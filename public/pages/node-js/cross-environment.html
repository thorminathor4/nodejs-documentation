Node.js - Cross Environment

The "cross-env" module provides the ability to run platform specific commands across all platforms.

Install Cross-Env

See how to install modules here: NPM & Projects.

Use Cross-Env

When you have installend Cross-Env, you can execute a command from another platform.
This is done by making a script in the package of a project, where "cross-env" is prepended to the command of the script
(Here the script "start" uses "cross-env" to run the command following "cross-env"):

"scripts": {
  "start": "cross-env PORT=3000 node app.js"
}


The script above sets an environment variable. Read more about that here: Use Environment Variables.
