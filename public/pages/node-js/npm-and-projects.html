NPM & Projects

NPM is the Node Package Manager and is included in the installation of Node.js.
NPM is used for developing Node.js projects.


Here are some things you can de with NPM:

Check The NPM Version

You can check which version of NPM you have with this command:

npm -v


Initialize a Project

You can initialize a project inside a folder with this command:

npm init


The command will ask you to specify the following, but none of them are required:
  • Package Name
  • Version
  • Description
  • Entry Point
  • Test Command
  • Git Repository
  • Keywords
  • Author
  • Licence
When a project is initialized, it will have a file named "package.json".
This file will have info about and ressources for the project.

Install a Module

You can install a module for a project and save it with the following command
(A project will be initialized if not alredy initialized, when you install a module)
(This command will install the "express" module. Replace "express" with an other module name, to install that module):

npm i express --save


When you install a module, it will be added as one of the project's dependencies.
See how to import modules here: Importing & Exporting Modules.
To see how to make an express server, go to this guide: Create a Server.

Install All Dependecies

If you are inside a project folder, you can install every dependency it has at once with this command:

npm i


Run Scripts

You can also make scripts in the package that you can run.

Here is a guide on how to make and run a script:
  1. Open "package.json" and set the "scripts" property to an empty object, if the "scripts" property is not already there
    ("..." is where the other properties of the package can be):

    {
      ...
      "scripts": {

      }
      ...
    }

  2. Make a new script by doing the following:
    1. Making a property in the "scripts" object with the name of the script in the "scripts" object
    2. Setting the value of it to a string that has the command, the script scould run
    (Here the script "start" is added to the "scripts" object to run the command "node app.js"):

    "scripts": {
      "start": "node app.js"
    }

  3. Execute the script in the terminal with this command
    (Here the script "start" is executed):

    npm run start
